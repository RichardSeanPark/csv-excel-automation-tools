How to Batch Rename Files by Date on Windows

If a folder contains many receipts, invoices, photos, or exports, renaming them one by one is slow and error-prone. A safer approach is to create renamed copies first, then review the result.

Recommended Workflow

  1. Create a backup or use copy mode.
  2. Choose a consistent date format such as YYYY-MM-DD.
  3. Add an optional label such as receipts, invoices, or products.
  4. Add a sequence number so files sort predictably.
  5. Review the renamed folder and the rename log.

Example

Before:

  • invoice_20260105.pdf
  • receipt-20260106.png
  • memo 2026-01-07.txt

After:

  • 2026-01-05_receipts_001_invoice.pdf
  • 2026-01-06_receipts_002_receipt.png
  • 2026-01-07_receipts_003_memo.txt

Why a CSV Log Helps

A rename log shows the original name, the new name, and whether each file was copied or skipped. This is important when files have similar names or when some files have no date.

Beginner-Friendly Option

Batch File Renamer is a local Windows app that uses copy mode by default and creates a rename_log.csv file after each run.
