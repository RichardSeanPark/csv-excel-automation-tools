# Intake Form

Use this before accepting any paid order.

## Required questions

1. What repetitive task do you want to automate?
2. How often do you do this task?
3. How long does it currently take by hand?
4. What is your operating system? Windows, macOS, or Linux?
5. Please upload a sample input file with fake or sanitized data.
6. Please upload a sample expected output file, or describe the exact output rules.
7. Are the input columns and file format always the same?
8. How many files or rows do you usually process at once?
9. Do you need Excel `.xlsx`, CSV, JSON, PDF, or another output format?
10. Does this require logging into any website or using a paid API?

## Auto-accept checklist

Accept as a simple fixed-price order only if all are true:

- Input file exists.
- Expected output is clear.
- No login, CAPTCHA, account automation, or prohibited scraping is needed.
- One script can solve the task.
- The task can be tested locally with the sample file.
- The buyer agrees to one minor revision within scope.

## Ask for separate quote if

- The task involves browser automation.
- The source data is from a website.
- The task uses a paid API.
- The output requires complex Excel formatting.
- The buyer needs a desktop app or web app.
- The buyer needs recurring hosting or monitoring.

## Reject if

- The request involves spam, fake accounts, mass messaging, CAPTCHA bypass, or credential misuse.
- The buyer refuses to provide sample input.
- The buyer asks for unlimited revision.
- The buyer cannot describe the desired output.
